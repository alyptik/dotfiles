NAME
----
zplug-clear - Remove the cache file


SYNOPSIS
--------
[verse]
'zplug clear'

    The word 'package' refers to the string in the format: "username/reponame".


DESCRIPTION
-----------
Remove the cache file.


OPTIONS
-------
*(None)*::
    no options


EXAMPLES
--------

Remove the cache file:

------------
$ zplug clear
